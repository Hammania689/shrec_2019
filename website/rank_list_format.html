Rank list format for SHREC 2018 - 2D Scene Sketch-Based 3D Scene Retrieval

The participants should submit one separate rank list file for each 2D scene sketch in the query set. The file must be an ASCII file; the filename should be the same as the query model and does not contain any file extension.

Each line of the file should contain a 2D sketch-3D scene model distance in the order of increasing dissimilarity and the index number of a retrieved target model. The distance and the name should be separated by white space.

Example rank list format for the query sketch 1.png:
The filename of the rank list should be 1 with no file extension.
The content of the rank list should be as follows:

The files should be collected in a single archive file and the name of the archive file should indicate the retrieval method. The archive file should be sent via e-mail juefei.yuan@gmail.com. If the participants have more than one method, they should submit an archive file for each method (the filename should indicate the method). Example archive file for the method X should have a name method_X.zip or method_X.rar. The file should contain either 70 (test) or 250 (all) rank lists of the query scene sketches.

If you have any question, please email to juefei.yuan@gmail.com
